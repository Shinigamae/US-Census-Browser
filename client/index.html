About the 2010 Census Browser

The Census Browser application is a free and open sourced application built using PhoneGap (http://phonegap.com). All data used within this application is freely available within the public domain, from http://2010.census.gov. In particular, this application uses the Demographic Profiles data set.

You can read more about this application, and how it was built at http://www.tricedesigns.com/2010-census/
